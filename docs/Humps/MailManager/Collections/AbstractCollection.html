AbstractCollection

class AbstractCollection implements IteratorAggregate, Countable, Jsonable, Collection, JsonSerializable

A simple abstract class for allowing access to a collection of objects. This is not intended to be a comprehensive Collection class, but rather, a convenient abstract wrapper for ensuring that the correct object arrays are being passed in to functions which type hint child classes.
