Collection

class Collection implements IteratorAggregate, ArrayAccess, Countable, Jsonable

A simple abstract class for allowing access to an array of objects. This is not intended to be a comprehensive Collection class, but rather, a convenient abstract wrapper for ensuring that the correct object arrays are being passed in to functions which type hint child classes.
