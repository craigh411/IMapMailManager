array searchMessages(string|array $criteria, int $sortBy = SORTDATE, bool $reverse = true, int $options)

Search for a message by the given criteria. The criteria can either be a string, such as 'ALL' or an array Where the key is the criteria and the value is a string or array of search terms.

see imap_search criteria (http://php.net/manual/en/function.imap-search.php)

It's important to note that this performs a logical 'AND' operation, so ['to' => ['foo@bar.com','bar@baz.com']] would return emails to all listed recipients, not any.

Parameters

string|array $criteria The search criteria (e.g. 'FROM' for 'from' email address). see imap_search criteria (http://php.net/manual/en/function.imap-search.php) This parameter can also be passed as an array for criteria with arguments e.g ['from' => 'me@example.com', 'to' => 'you@example.com'] and each key can also be passed as an array (e.g. ['to' => ['me@example.com', 'you@example.com']]).
int $sortBy The criteria to sort by (see: http://php.net/manual/en/function.imap-sort.php)
bool $reverse Whether the sort should be in reverse order
int $options Any options to pass through the imap_sort(see: http://php.net/manual/en/function.imap-sort.php)

Return Value

array An array or message numbers
