ImapMessageCollection searchMessages(string $criteria, int $sortBy = SORTDATE, bool $reverse = true, int $messageOptions, bool $headersOnly = false)

Search for a message by the given criteria. The criteria can either be a string, such as 'ALL' or an array Where the key is the criteria and the value is a string or array of search terms.

see imap_search criteria (http://php.net/manual/en/function.imap-search.php)

It's important to note that this performs a logical 'AND' operation, so ['to' => ['foo@bar.com','bar@baz.com']] would return emails to all recipients, not any.

Parameters

string $criteria The search criteria (e.g. 'FROM' for 'from' email address). see imap_search criteria (http://php.net/manual/en/function.imap-search.php)
int $sortBy The criteria to sort by (see: http://php.net/manual/en/function.imap-sort.php)
bool $reverse Whether the sort should be in reverse order
int $messageOptions Any options to pass through the imapfetchbody (see: http://php.net/manual/en/function.imap-fetchbody.php)
bool $headersOnly Return headers only, don't fetch the message body.

Return Value
